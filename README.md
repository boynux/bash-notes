Bash note taking tool
=====================

Add `notes` to your `bashrc` or `zshrc` file.

Usage:
------

To create or edit a note:

$ n note-name

To list notes:

$ nls

To output note content on terminal:

$ npr note-name

To show MarkDown text using (mdless)[http://brettterpstra.com/projects/mdless/]

$ nless note-name
